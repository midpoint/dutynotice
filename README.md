# 学校值班提醒程序

学校各类值班安排查询工具。

## 功能

- 查看今天/指定日期的所有值班安排
- 查看本周或指定周的值班安排
- 查询某人的所有值班日期
- 支持日期范围查询

## 文件说明

```
dutynotice/
├── config.py      # 配置文件（学期时间、值班人员名单）
├── dutynotice.py  # 值班逻辑
├── cli.py         # 命令行界面
└── requirements.txt
```

## 使用方法

```bash
# 查看今天值班
python3 cli.py today

# 查看指定日期
python3 cli.py date 2026-3-5

# 查看本周或第N周值班
python3 cli.py week
python3 cli.py week 3

# 查询某人所有值班
python3 cli.py person 郭仕基

# 查看日期范围
python3 cli.py range 2026-3-2 2026-3-8

# 帮助
python3 cli.py help
```

## 配置说明

修改 `config.py` 文件可自定义：

- 学期起止时间
- 夜间值班分组
- 行政值班人员
- 周日行政值班人员

## 值班规则

| 类型 | 时间 | 安排 |
|------|------|------|
| 夜间值班 | 周日到周四 21:30-次日7:00 | 3组循环，每周1组，每天1人 |
| 行政值班 | 周一到周五 7:00-21:40 | 每天固定人员 |
| 周日行政值班 | 周日 18:00-21:40 | 每5周循环 |
